# Changelog for @gauzy/ui-core

## [Unreleased]

### Changed

-   Updated `gauzy-logo` component to reflect new logo branding.
-   Modified `header`: added navigation tracking in header component to update selectors on route change.
-   Added styling for asterisk (\*) to visually indicate required fields in forms.
-   Added support for tracking weekly limit hours.
-   Implemented tracking of removal or unassignment per task/project in the web time tracker.
-   Added time limit tracking for adding and editing time logs.
-   Added a new reusable time input component.
-   Improved table filters functionality and fixed bugs in the grid layout.
-   Added new pipes and modified existing ones.
-   Modified PDF and CSV export handling.
-   Created new directives.
